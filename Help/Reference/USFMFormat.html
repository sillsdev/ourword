     USFM File Format for Scripture

OurWord supports SIL's "standard format," in two flavors:

  • Shoebox/Toolbox, described here.
  • Paratext's USFM_DRAFT.STY (a subset of USFM) plus a few additional markers

A full description of USFM is maintained by the United Bible Societies at this Internet site.

Paratext introduced the USFM_DRAFT stylesheet in order to provide a subset of USFM that would be suitable for drafting. OurWord uses most of the markers in USFM_DRAFT. When a translation project reaches the stage of being published, it is likely that far more formatting will be desired than can be accomplished with OurWord's limited set of markers.

USFM Markers that are fully supported in OurWord:

  • \id - File identification
  • \h - Running header
  • \mt - Major Title (level 1)
  • \st - Subtitle (commonly \mt2 in USFM.)
  • \ms - Heading - major section level 1
  • \mr - Heading - major section range references
  • \s - Section level 1
  • \s2 - Section level 2
  • \r - Parallel references
  • \c - Chapter number
  • \v - Verse number
  • \p - Paragraph
  • \q - Poetry (level 1)
  • \q2 - Poetry (level 2)
  • \q3 - Poetry (level 3)
  • \qc - Poetry (centered)
  • Footnotes: \f .... \fr...\fr* \ft....\ft* .....\f*
  • Cross References: \x ..... \xo ... \xo* \xt ... \xt* ....\x*

Markers that are transformed:

  • \mt1 (Major Title level 1) is converted to its equivalent, \mt
  • \mt2 (Major Title level 2) is converted to the Toolbox equivalent, \st
  • \s1 (section level 1) is converted to its equivalent, \s
  • \m (Paragraph at margin, no first line indent) is converted to \p
  • \nb (Paratext text, no break from previous paragraph) is converted to \p
  • \q1 (Poetry level 1) is converted to its equivalent, \q

Markers that are not currently supported in OURWORD:

  • \imt - Introduction major title level 1
  • \ip - Introduction paragraph
  • \qs - Poetry text: "Selah"
  • \b - Poetry stanza break (blank line)
  • \sp - Heading - speaker (e.g., "Job")
  • \d - Heading - descriptive title / Hebrew subtitle
  • \fq - Footnote quotation or alternate rendering
  • \fig - Figure / Illustration / Map

On import, OurWord automatically detects the format and reads it correctly. OurWord saves files in the Toolbox standard; but you can export to USFM via the Export command in the Configuration dialog (in a translation's properties page).
