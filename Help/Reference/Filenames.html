OurWord Help
     File Names

OurWord controls the name of the file. The file name is composed of information about the book, and the name changes as this information is changed. (For example, if the book's stage is changed from "Draft" to "Team Check", the filename will change to indicate the new stage.) The filename is composed of:

LangAbbrev-BookAbbrev-StageAbbrev-Version-Date.db

where:

  • LangAbbrev is typically the Ethnologue 3-Letter code, or some other shortened abbreviation of the language name. It is set in the Translation Properties Dialog.
  • BookAbbrev is typically the standard 3-Letter SIL/UBS book abbreviations, but short abbreviations in other languages are also supported.
  • Stage is one of the stages as defined in Tracking a Book's Stages of Translation.
  • Version is a editing major pass, of the form 'A', 'B', 'C', etc. When changes are made in a pass through an entire book, a Version should be used.
  • Date is the date the file was last modified by OurWord.

For example, the filename Ama-LUK-Adv-B 2008-01-17.db would be interpreted as:

  • Ama - The Amarasi language
  • LUK - Luke
  • Adv - Advisor Check Stage
  • B - Version B
  • Last edited on 17 January 2008

When importing a book, the Import Dialog provides the opportunity to declare the book's stage and version information. Subsequent saves will result in the file being saved to the appropriate filename that is generated from this information.
