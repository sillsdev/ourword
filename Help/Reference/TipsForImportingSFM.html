     Tips for importing Standard Format Toolbox Scripture Files

Use the following process to import files in Toolbox Scripture Standard Format into OurWord:

1. Check (e.g., in Shoebox or Toolbox) to see if the file to be imported has the same number of records as the Front Translation. If not, work through the \rcrd and \s fields until you get them to match. The simplest way from within Toolbox is to activate one of the files by clicking on the top bar, and then look down at the bottom bar where it will tell you how many records the file has (e.g. 11/96 means you are looking at record number 11 out of 96 total records in the file). Then click on the other file and see if it has the same total number of records.

2. Try to import the file into OurWord.

3. If it doesn't accept it, OurWord should give a message alerting you to the (first) reason encountered. (If OurWord does not give a message, this is a bug and should be reported. Please email both the Front and the Target files.)

4. When a file is newly accepted into OurWord, visually skim through each record.

  • mismatches in paragraphing will be fairly obvious
  • mismatches in placement of footnotes will be fairly easy to spot
  • text not yet entered will be fairly easy to spot

5. Jot each reference on a piece of paper, with a shorthand note of the type of fix needed

6. Close OurWord, and open the file back up in Toolbox, side-by-side with the Front Translation file. Go straight to the references noted and clean them up there.

This approach is significantly faster than going meticulously through a whole file carefully looking at all the SFM codes to see that they match.

Updated as of Version 1.0.
