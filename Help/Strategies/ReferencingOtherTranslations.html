     Referencing Other Translations

The main display of OurWord is to show the source/front translation on the left, and the draft in the target language on the right. But often translators want to refer to additional translations; and for that reason OurWord supports the display of reference translations.

You first set up reference translations in the Tools-Configuration dialog. There, under the Reference button, you specify these translations in the same manner that you do for the front and target languages under the Essentials tab. Thus you supply the name of the translation and the filenames for the various books within it.

Once there are one or more reference translations defined for a given project, then under the Window menu the "Show Reference Translations Pane" item will be visible. Clicking on that item reveals the Reference Translations pane. There, as you move around in the draft, the corresponding paragraph is shown in each of the translations that you have defined.

OurWord attempts to only show the paragraph that corresponds to the current context. Sometimes this is difficult to calculate, and thus multiple paragraphs are shown in the reference translation. This is not a bug; it simply reflects the lack of structural correspondance between the target and the reference translations.

Updated as of Version 1.0.
