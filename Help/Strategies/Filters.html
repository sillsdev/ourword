OurWord Help
     Only Show Sections That... (filters)

The "Only Show Sections That" Dialog is the means by which you can restrict the sections that you are working with to a smaller subset, based upon some criteria you set. OurWord provides a number of criteria that you can choose from. (Software such as Shoebox/Toolbox and FieldWorks refer to this type of feature as a filter.)

The dialog is accessed from the Tools dropdown. By default, this option is not available; you must first turn it on through the Setup Features Dialog.

This dialog is likely to change as new types of criteria are added, but as of Version 1.0 the available choices are:

A Word or Phrase...

  • In the Vernacular - Examines sections for a word or phrase in the translation, whether in the Scripture text, picture caption, or footnote.
  • In the Front - Examines the Front Translation for the word or phrase. (Note this is the front translatin's scripture, not its back translation.)
  • In the Back Translation - Examines the target's back translation for the search text.

    Note that if you want to find something in the vernacular, that string must be typed in the box for the vernacular; if you want to find something in the front translation, that string must be typed in a different box - the one for the front translation. Etc.

A Note of Type...

  • To Do - Checks for the presence of a "To Do" note, used to flag work that needs to be done in the section.
  • Ask UNS - Checks for the presence of an "Ask the Uninitiated Naive Speaker" note, used for checking and testing.

Possible Problems...

  • Untranslated Text - Checks for areas where there is text in the Front, but only a blank in the Target. You can use this to find all of the sections where drafting is still needed.
  • Mismatched Quotes - Examines quotes (which must be of the angle bracket notation (e.g., <<Hello>>.) Checks to see if there is a mismatch where, for example, there is no closing quote to correspond to an opening quote.
  • Punctuation Problem - Checks for the following types of issues related to punctuation:
    • No end-punctuation at the end of a paragraph
    • "Orphaned" punctuation; that is, punctuation that is not adjacent to a word. (For example, a space prior to a period.) E.g.,
      • "Space at the end of a sentence ?"
      • "< Space just after a quote begins"
      • "Punctuation with spaces on both sides : like this colon"
      • "Two or more orphaned punctuation together !>"
    • End-Punctuation (such as a period or question mark) following a footnote, rather than preceding. it.
    Punctuation should be defined on the writing system (via the Properties dialog.) Otherwise, OW uses defaults that are appropriate for many roman writing systems (such as English.)
  • Picture Cannot Be Located on Disk - Checks for the picture's path name to find those picture which do not have a file on the disk. (Note that you will need to exit OurWord and use an editor such as Toolbox in order to address any problems found. Also, the picture may display while in OurWord, because OurWord uses the path as found in the Front Translation to display the picture; but with this error, it is the Target Translation's path which is examined.)

Structure...

  • A Picture with a Caption - Looks for those sections in which the picture in the Front Translation has a caption. Use this to check the consistency in the way you have handled your captions.
  • A Quote Paragraph - Looks for sections that contain one of the several types of quote paragraphs. In standard format, these are "\q", "\q2", "\q3", and "\qc".

The "This Section Qualifies if" control allows you to choose one of two options:

  • Any One of These are True - This option means that if any single one of the tests passes, then the section is included in the results. Thus for example, you could say "Show me all sections that either havea Mismatched Quotes or Untranslated Text."
  • All of These are True - This option means that every one of the checked tests must pass in order for the section to be included. An example would be "Show me all sections that have both a To Do Note, and the word "Farisi" in the vernacular."

Click on the Turn On button to instruct OurWord to examine the sections for the requested criteria. This will remain active until you go back to the dialog and click on the Turn Off button. Thus once this has been Turned On, clicking on the right arrow icon will take you to the next section matching the criteria you have set.

Updated as of Version 1.0.
