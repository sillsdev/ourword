     Restore From Backup

The Restore From Backup Dialog is used to restore the translation of a book to a previous version. The Auto-Backup command must have been enabled (see the Automatically Backing Up Data topic.) The dialog is accessed from the Tools dropdown, provided that the feature has been turned on via the Setup Features dialog.

The dialog shows the name of the book (and its current filename) that will be restored. This it the book which is currently being edited; and thus the one that will be replaced.

To restore to an older version,

  • Click on the Browse button. This will bring up a list of the previously backed-up files that are available for the restore operation. Chose one from this list and click on OK. (The file will appear in the white area of the dialog.)
  • Click on the Restore button.
  • You will be presented with an "Are You Sure?" dialog, to make certain that you really want to proceed. Keep in mind that a restore operation will permanently erase your current work, replacing it with what is in your backup file. Therefore press "Yes" only if you are sure you want to proceed with the restore.
