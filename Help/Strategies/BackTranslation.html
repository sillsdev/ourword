     Back Translation

The Back Translation window supports entering of a prose back translation. (An interlinear format will likely be supported in a later version of OurWord.) In the Back Translation window, the translation appears on the left-hand side, and the back translation is then entered into the right-hand side.

When you set up the translation, under the Configuration dialog's General tab there is a place to enter a "Advisor Text" writing system. This is where you indicate the writing system that is used for the back translation (and other areas where advisor notes are entered.)

The Back Translation window must be enabled through the Tools-Setup Features dialog. By default it is turned off, as we anticipate that a new user will start by drafting, and does not need the confusion of seeing this in his user interface.

Updated as of Version 1.0.
