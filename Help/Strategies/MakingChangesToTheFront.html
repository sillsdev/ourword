OurWord Help
     Making changes to the Source / Front translation

OurWord does not currently offer a feature for making global changes. These are changes which apply to the Front Translation and simultaneously to all daughters currently in progress. These include:

  • Changing the section breaks (e.g., splitting a long section into smaller ones, or moving a verse from one section to another)
  • Adding or removing pictures
  • Adding or removing global footnotes or cross references
  • Changing paragraph breaks
  • Changing paragraph styles (e.g., changing a paragraph from "p" to "q".)

A future version of OurWord will hopefully support these kinds of changes, but in the meantime they must be done by hand. This means that you must load each DB file into an editor (such as Toolbox or Paratext) and make the changes there; then run OurWord to make certain that everything works as expected.

OurWord has expectations about the standard format that it will accept, and error messages are presented if you attempt to load a book which does not meet these expectations. It is easy to introduce errors when working in a free-form editor such as Toolbox, so we recommend that you minimize your use of such programs. I am hopeful that future versions of OurWord will make outside editors less necessary.

When running OurWord, if you notice that paragraphs are not lining up properly between Front and Target, that is a clue that there is a mismatch in the underlying files. It may be something as simple as one paragraph being a "p" while its corresponding one is a "q". Or it may be that the picture is not following the same verse between the two versions. It will be helpful in such a case to use side-by-side windows in Toolbox to compare the two, in order to discover what has changed.

If you are stumped, feel free to email me the offending files and I'll be happy to take a look. I'll need both the Front and the Target files.
