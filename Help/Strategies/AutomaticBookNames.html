     Automatic Book Names

OurWord allows you to set up book names for automatic translation in cross references and parallel passages. Once set up, every time a book name is encountered in the source translation, it is automatically translated into the correct book name in the target translation. This automation handles what is otherwise a very error-prone process.

In the Tools-Configure dialog, under the Essentials tab; both the Front and the Target languages have a "Book Names" section.

Thus for the Front, you enter the name of Genesis in that front language; and for the Target, you again enter the name of Genesis in that target language. Thereafter, when the a \r or a \cf field includes the front's name for Genesis on the left-hand side of the drafting window, it will be given the target's name on the right-hand side.

Updated as of Version 1.0.
