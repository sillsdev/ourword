OurWord Help
     Pictures

OurWord recognizes most types of picture format (as supported by Microsoft's DotNet framework. Pictures are displayed within the drafting window, so that the translator can see them and enter a meaningful caption.

Pictures are entered into the standard format file as described here.

Pictures are often stored in a single place on the local hard drive. In the Tools-Configuration dialog, under the Options tab, there is a setting "Folder to search for pictures" that you can use to specify the root folder for your pictures. When OurWord comes across a picture's file name, it will search that root folder and all of its subfolders in an attempt to locate the file. If found, OurWord will then display it; otherwise it displays a message that the picture could not be found.
