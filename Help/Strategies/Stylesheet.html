OurWord Help
     Stylesheet

OurWord uses a small, standard set of styles for paragraphs and characters. Final typesetting will probably require additional styles, whereas OurWord presents those which are typically used during drafting.

Styles are stored in the TeamSettings file. This allows you to set a style once, and then use it for all of the teams working in the language cluster.

Styles are edited in the Configuration Dialog, under the Team Settings group on the left sidebar.

The various styles are shown in the list to the left; when you click on one of these you will see its individual settings to the right, where you can make changes to them. In the list, "P" represents a paragraph syle (which has settings for both characters and paragraphs), and "C" represents a character style (such as a chapter number).

For character settings, note that there are different font settings for each writing system. This permits you to choose a font and font height that works best for a given writing system.

The Restore Default Values button returns the entire stylesheet to its original factory settings.

Updated as of Version 1.0.
