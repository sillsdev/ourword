OurWord Help
     Creating a new project

A Project consists of the top-level settings that have all of the information OurWord needs to work with a translation. It includes identifying the source translation, specifying the writing systems, describing the style sheet, and numerous other settings.

OurWord provides a wizard to walk you through the most essential settings that it needs in order to create a new project. You access this wizard through the Project-New Project dropdown menu. Once these settings have been supplied, the new project is created. You can then go through the Tools-Configure dropdown menu in order to fine-tune these settings, or to enter more advanced options.

The various settings that are available for a project are the subject of the majority of the topics in this help documentation. Refer to the Table of Contents for further help on these individual topics.
