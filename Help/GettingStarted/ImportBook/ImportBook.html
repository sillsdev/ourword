OurWord Help
     Importing an existing book

OurWord currently supports two flavors of Standard Format: (1) the format used in Shoebox/Toolbox, and (2) a subset of USFM that is used by Paratext.

Starting the Import Process

Import is initiated via the Import button in the Configuration dialog. This button first brings up a wizard by which you supply basic information about the file:

The first step in the wizard is for you to browse for the file that you wish to import. After you navigate to the file, the wizard does a quick scan of the file to determine whether or not it is one which can be imported, and displays information (or error messages) as appropriate. Only after a suitable file has been supplied with the NEXT button become enabled, so that you can proceed to the next page in the wizard.

The second step is for you to identify the book being imported. This is a matter of choosing one of the possible 66 books from a list. OurWord will only show those books that are possiblities, thus if you do not see your book in the list...

  • The book may have already been imported
  • If you are importing a book into your translation, that book must already be defined in the Front translation; if it has not yet been defined there, then it will not appear in this list.

The third step gives you the option of supplying additional information about the book. For the Stage, you can indicate whether the book is, e.g., a draft, or has been back translated, or has been consultant checked, etc. For the Version, you can append an optional letter to indicate, e.g., how many passes have been made through the draft. (There is more information about Stages and Versions here.)

The fourth step in the wizard allows you to specify the folder where the file will be stored. OurWord has its own naming system for filenames, through which you can see at a glance the book's language, abbreviation, stage, and version. OurWord does not give you control over the filename, but does let you decide into which folder you want it to be placed. You will normally want to store all of the books of a translation together on your hard disk.

Once this information has been entered, the wizard displays a summary page of the decisions you have made. Clicking on Finish will ccause OurWord to proceed with the import.

Running into problems

It is very common for a book which was created by other software to not import. Standard Format can be a slippery animal, and mistakes in the use of markers, or in their order, or numerous other items can result in an error condition. In this case, OurWord will complain of the error, and will put up an editor by which you can attempt to solve the problem. (Or alternatively you can exit OurWord and do the fix in other software.)

The following error messages may appear when importing:

  • Bad Chapter Number
  • Missing Paragraph Marker
  • Missing Paragraph Marker for \cf field
  • Missing Paragraph Marker for Note
  • Missing Verse Number
  • Missing Verse in Section
  • Structure Mismatch
  • Structure Section Miscount

Examine the Results

Unfortunately it is not always possible to correctly import these files, because the files often have errors and unrecognized markers within them; so you are strongly encouraged to carefully review the book once it has been imported into OurWord to make sure that everything appears correctly. If not, you may wish to edit the book in your external editor, and then do the import again.
