     Import Error: No Verse in Section

This error occurs when there is a section that does not have any \v markers in it.

An example of this error:
     

While it is technically possible to publish Scripture with very small sections that do not have verse numbers, OurWord does not permit it due to internal book-keeping it must do. So if you are seeing this error because you desire very small sections, you will unfortunately need to make your sections larger.

On the other hand, you may be seeing this error because you are missing some \v markers, in which case all you need to do is add the missing markers.

Possible cures:

  • Consider making the section cover additional paragraphs, so that a verse number will be present.
  • Add any missing \v markers.

After you make the correction, click on the Try Again button to instruct OurWord to resume importing the book; or alternatively you can click on Cancel to abandon the import and fix it in some other editor.
