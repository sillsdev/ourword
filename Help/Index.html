Welcome to OurWord, software designed to support Bible translation by nationals who may have had little, if any prior computer use. If we have suceeded in our useability goals, then translators will not need to use this help system; thus the following modules are designed for the Advisor who works behind the scenes to set up OurWord and make the translator successful.
